# Notes

* Software Discovery Tool Annual Review
    * Elizabeth Joseph and Tushar Biswas presented the annual review for Software Discovery Tool
* Mainframe Infrastructure Status
    * Mike Friesenegger provided an update on the mainframe infrastructure from the perspective of the Feilong project.  
    * Friesenegger said the Feilong community has successfully been granted access to the Open Mainframe Project infrastructure.
    * Friesenegger added that infrastructure onboarding is a work in progress, but the environment is functional and a second-level ZVM environment has been successfully started and accessed.
